AIDOC‑AP — Application Profile for AI System Documentation

A machine‑readable representation of the technical documentation the EU AI Act requires for high‑risk AI systems.

AIDOC‑AP is an application profile that structures the technical documentation obligation of the EU AI Act (Article 11 / Annex IV) as an OWL ontology, reusing established vocabularies where possible. It is developed with an LLM‑assisted, competency‑question‑driven pipeline and validated on real pilot systems. Explore the resource below.
